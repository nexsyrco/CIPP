# Deleted Sites

This page will display the deleted SharePoint sites

## Table Details

The properties returned are for the SharePoint PowerShell command `Get-SPODeletedSite`. For more information on the command please see the [Microsoft documentation](https://learn.microsoft.com/en-us/powershell/module/microsoft.online.sharepoint.powershell/get-spodeletedsite?view=sharepoint-ps).&#x20;

## Table Actions
