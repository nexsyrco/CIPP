# Copilot Settings

This is a simple table view of Microsoft 365 Copilot tenant settings for the selected tenant.

## Table Details

| Column  | Description                                                                         |
| ------- | ----------------------------------------------------------------------------------- |
| Setting | The name of the Microsoft 365 Copilot tenant setting.                               |
| State   | The setting's current configured state (e.g. Enabled, Disabled, or Not configured). |

## Table Actions

<table><thead><tr><th>Action</th><th>Description</th><th data-type="checkbox">Bulk Action Available</th></tr></thead><tbody><tr><td>Set Status</td><td>Changes the selected setting's state. Prompts for a desired state: Enabled, Disabled, or Not configured. Refetches the list so the updated state is reflected.</td><td>true</td></tr></tbody></table>
